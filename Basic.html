TextStyles

HTML(HyperText Markup Language)is the most basic building block of the web.It defines the meaning and structure of web contents.other technologies besides HTML are generally used to describe a web page's appearance(CSS)or functionality /behavior(JavaScript).

The HTML style attributes is used to add styles to an element,such as color,font,size,and more

BoldThe"b" tag in HTML is used to specify the"b" tagdisplay in bold size.You can do that by using font-weight:bold;property in CSS .it is a container tag that contains an opening tag,content & closing tag.

ItalicsThe"i" tag defines a part of text in an alternate voice or mood.The content inside is typically displayed initalic.The "i" tagis often used to indicate a technical term,a phrase from another language,a thought,a ship name,etc

UnderlineTHe"u" tagin HTML stands forunderlilneand it's used to underline the text enclosed with in this "u" tag.This tag is generally used to underline misspelled words.This tag requires a starting as well as ending tag.
